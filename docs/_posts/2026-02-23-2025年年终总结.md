蛇年走，马年来。写在春节假期的最后一天，本文主要是和 2025年的自己告别，展望下2026年的新生活。

<!--more-->

## 工作

工作方面在国企来来回回总的来看还是有一些些的收获的。从技术、职场、心态来讲讲。

技术上今年总算有机会做了些算法的工作，任务上是围绕着文本识别和关键信息抽取来做的。在这里我一个人整体从 0 开始借助开源的力量搞定了 VLM 从标注、训练和部署的全链路大大小小的数据算法工程等相关一系列工作。这部分别看网上自媒体吹的很简单的样子，实际脏手做的时候还是会遇到非常非常多的问题。可以说媒体中大家以为的关键信息抽取的简单方案完全做不到 95+% 的商业可用标准，实际上90+%都很难保障。除此之外，让我觉得有趣的还有高并发的时候大模型的推理其实因为很多原因会存在不稳定的情况，这些都是做之前完全想不到的点。以上所有的算法的经验都沉淀在了脑子还有一些个人的脚手架中（如 Textify，围绕 Label Studio 标注平台的各种小工具、VLM 的训练代码、量化代码 etc）。除此之外，这两年也是蛮深刻的意识到软件并不比盖楼简单，不是过去小小的大厂中学到的那点算法工程所能 cover 的，想做一个稳定好的解决方案交付算法只是大厦中的 10%，脱离了大厂的基建之后有太多的「脚手架」需要补充，这可能是不从事这个行业意识不到的软件壁垒。好在有 AI 加上一点点学生时代打下的基础，补充补充也是够用的。这部分的知识真的很杂，从 ELK、Spring、VUE、到 Kafka / Redis / Docker / Nginx / Nacos 等等等，今年得到了很多实战的知识补充。

职场上说实话意识到的最大的收获在于「不要在职场上交朋友」，职场更像是一个丛林在国企更是内部的信息差相比大厂要更严重。大厂可以不干了就走，但是国企大部分人是做不到的。因此这里的职场更像一个村子，一个盘根错节的村子，一个贯穿着各种能讲不能讲的事情的村子。好在自己的家庭收入中工资的占比已经开始下降了，老婆的职业看起来也蛮顺利的，所以脱离村子或者远离打破脑浆的核心地带应该是可以做到的。潜龙勿用也许是这里的最佳的行为准则。

心态上说实话今年好了一些，不过还是时不时会将自己的工作评价体系交到外人手上带来心态上的烦躁。这点在下半年满满的的到了缓解，心态愈来愈像一个专业的职场人，而不是一个充满学习心态的学生。这不意味着我躺平了，而是说我学会了一种伪装，本质还是好奇、脏手的实践派，但其他人别想纯纯白嫖我的劳动经验，我也不再会无偿的分享。别人也许会说这是一种成熟，但我只会说这是一种妥协。

## 投资

投资上在好友 Y 的带领下快快乐乐度过了两年，即将步入第三年。今年通过翻阅笔记来看，投资相关有意义的笔记停留在 5月份，从 6 月开始就没有特别深度思考带有预测性质的记录了。这让投资框架的迭代缺失了很多对照项目从而无法进行迭代，这点在今年就要非常注意。决策反过来来看犯下的最大的错误在于美团上，我错估了其商业模式的壁垒，视野纯纯聚焦在美团和阿里的 PK 之上，而忽略了最大的流量来源头条在到店业务的趁火打劫。这导致美团在每次被挑战的时候都面临着艰难迎敌的同时，还带来盈利端被偷家的可能性，要是早些看清楚这点的话也许可以不会亏甚至微利跑掉。投资能力方面，今年确实着手花了一些时间去做了各种各样的事情，但是大都也是虎头蛇尾所以没有什么好总结和展望的，只能说今年不要再做这种不收尾的事情。

## 习惯

生活习惯上除了健康的作息还是没有养成，身体指标在下行。另一个意识到自己很不好的一个习惯是做了很多事情但是不收尾。反观今年，借着尝试 Vibe-Coding 的旗号尝试了自动播客 / 量化分析 / 13f-tools / ValueAns 等等等一系列事情，但没有一件事情是收尾的。整体都是做到了 0-1，但是离可用还有临门一脚的状态，这点非常不好。所以，今年对自己的要求就是开的坑一定要填掉，如果不填掉那就不要开坑。个人的猜测，如果做到这点的话 2027 年反看 2026 年的时候会得到一个非常好的观感，会更加接近理想中的自己。
